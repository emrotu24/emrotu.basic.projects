How to Download and Play the Game

1. Click the download link provided on this site

It will take you to the GitHub Releases section.

2. Find the file named build03.zip

Scroll down to the "Assets" section of the release and click on build03.zip to start the download.

3. Extract the ZIP file

Once the download is complete, right-click the file and choose "Extract All…" or use any unzip tool to extract the contents.

4. Open the game folder

After extraction, open the folder that contains the game files.

5. Run the game

Double-click the executable file "My project" to start playing.
